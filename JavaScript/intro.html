Introduction to JavaScript

JavaScript ― Это интерпретируемый язык программирования общего назначения с динамической типизацией данных.

Интерпретируемый значит что программа выполняется в исходном виде т.е., исходный код не преобразуетсяв машинный или промежуточный код. В связи с этим ошибок на этапе компиляции и компоновки отсутствуют, есть только ошибки на этапе выполнения. Интерпритатор JavaScript может быть установлен в любую операционную систему. Это дает хорошую переносимость кода или кроссплатформенность. JavaScript также иногда используют для разработки мобильных предложений под андроид. JavaScript также часто используют в тестировании для написания скриптов, но несмотря на все это чаще всего JavaScript используется для разработки web приложений, а именно FrontEnd части приложений. И следовательно основным и самым распостраненным интерпритатором JavaScripta является самый обычный браузер.

Динамическая типизация говорит о том, что тип переменных и констант определяется во время присваивания, по типу значения справа (r-value). l-value = r-value; Это в свою очередь означает то, что у переменной нет как такового постоянного типа данных, это достигается благодаря тому, что абсолютно все типы являются объектами а также благодаря полиморфизму.

Переменные и типы данных:

Переменные и константы

var, let и const
var
объявляет глобальную переменную, несмотря на область видимости, в которой она объявлена. Если записать значение в несуществующую переменную, то это не вызовет ошибку, это будет аналогично ключевому слову 'var';
let
объявляет локальную переменную, только для той области видимости, в которой она объявлена;
const
объявляет константу, константы могут быть только локальными;
