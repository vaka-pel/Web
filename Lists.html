Списки

Список определений

Позволяет создавать списки терминов с их разъяснением

<dl> ― Definitions List;
<dt> ― Description Term;
<dd> ― Description Details;
<dt> и <dd> всегда используются парой.
Нумерованный список
Позволяет нумеровать элементы текста. Это нужно для перечисления в строгом порядке.
Маркированный список
Позволяет перечислять элементы текста в любом порядке.
Список определений
Позволяет перечислять понятия и давать им определения.

Нумерованный список

<ol> ― Ordered List;
<li> ― List Item;
  1. List Item 1
  2. List Item 2
  3. List Item 3
  4. List Item 4
  5. List Item 5

  4. Продолжение первого списка
